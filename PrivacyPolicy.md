## 4. Information We Collect

### 4.1. User-entered data

The App collects information that users enter manually:

* Pet information (name, type, breed, age, etc.)
* Glucose levels and notes
* Insulin injections (type, dosage, notes)
* Meals (type, amount, notes)
* Other user notes related to pet health

**No automatic medical or biometric data collection** occurs.

### 4.2. Local storage

All data is stored locally on the user’s device by default.

### 4.3. Cloud synchronization (PREMIUM)

If cloud sync is enabled, data is stored in **Firebase**, associated with the user’s Google account. We:

* **do not store data on our own servers**
* **do not access the content** of user records
* do not share data with third parties, except for Firebase services required for synchronization

### 4.4. Technical data

The App may collect anonymized technical information:

* device type
* OS version
* crash reports
* usage statistics

This data is used only to improve the App and fix errors.

## 5. How We Use Data

User-entered data is used solely for:

* displaying diary entries
* generating charts and statistics
* creating local reminders
* generating PDF reports (PREMIUM)
* cloud synchronization via Firebase (PREMIUM)

The App **does not analyze medical data, provide diagnoses, or recommend treatments**.
